Motion Reactive Visuals

Using a motion sensor, this music video responds to the listeners physical interaction with the song. When the listener comes in close proximity with the sensor the colour/hue in the video changes colour.
I am interested in how audience members can directly influence their own experiences and interaction with the artist, creating a more stimulating experience.
I used an arduino and a motion sensor for the set up, the visuals were shot using milk and washing up liquid. The colour is added post by using a javascript code I wrote that reflects the distance the user is away from the sensor.
Test video to show how the motion sensor works is below.
